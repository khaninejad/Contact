Simple Laravel Contact Form
===================


This is a  basic contact form built in **laravel**

----------


Documents
-------------

Clone this git

> git clone https://github.com/khaninejad/Contact.git
> cd Contact
> composer update
> php artisan migrate

Edit .env file for mysql & mail driver

That's All
